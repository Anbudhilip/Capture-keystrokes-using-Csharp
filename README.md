# Capture-keystrokes-using-C#
## Description
Using WH_KEYBOARD_LL low-level hook to capture the keyboard keystrokes globally and log the strokes in Text file.
## Features
Detect key presses (eg.G ,Lshift , Ctrl)
Capture the keystroke and log the 
## Requirements
To set up and run this project, ensure you have the following:

#### Development Environment:

Visual Studio Code or Visual Studio (recommended).
.NET 6.0 SDK or later.
####  Dependencies:

Windows OS (required for low-level keyboard hooks).
Windows form (included with .NET).
####  Setup Instructions:

Clone this repository:
git clone [https:github.com/Anbudhilip/Capture-keystrokes-using-Csharp.git](https://github.com/Anbudhilip/Capture-keystrokes-using-Csharp).
Open the project in Visual Studio Code or Visual Studio.
Build and run the project.
